## Models
Here you can find all models, regarding the face depth reconstruction. 
1. The weights file "Attention_UNET_b5_e10ckp9_weights.pt" can be downloaded and used in the Google Colab, to use the pretrained model for face depth reconstruction
2. Regarding the MiDas pretrained model, please download the "Midas_DPT_b1_e10ckp9_weights.pt" weights and 
load them to the appropriate model, in the respective jupyter notebook
